# Titanic Survival Analysis
An exploratory data analysis (EDA) of the Titanic dataset using R and ggplot2.

## Analysis Overview
* **Data Cleaning:** Handled missing Age values using mean imputation.
* **Feature Engineering:** Categorized passengers into Age Groups and Survival status.
* **Statistical Testing:** Performed ANOVA to test the significance of Age and Fare on survival.

## Key Visualizations
* **Boxplots:** Comparing Fare distribution across survival status.
* **Pie Charts:** Visualizing the proportion of survival across age groups.
* **Scatterplots:** Examining the relationship between Age and Fare.
